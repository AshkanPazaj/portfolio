# Cafe X

A fictional neighbourhood café website with real photography, pickup ordering, table reservations, a promo carousel, and location call-to-action sections.

## Features

- Responsive landing page with hero and menu highlights
- **Order** flow for pickup (`order.html`)
- **Reserve a table** flow (`reserve.html`)
- Client-side form handling and validation

## Tech stack

- HTML, CSS, Vanilla JavaScript
- No framework, no build step

## Run locally

Open `index.html` in a browser, or:

```bash
python -m http.server 8000
```
