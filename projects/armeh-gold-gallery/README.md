# Armeh Gold Gallery

A luxury Persian (RTL) gold jewellery boutique site — collections, product detail modal, cart, user accounts, and admin panel.

## Features

- Homepage with collections grid, about, and contact
- Seven collection pages with filters (karat, name, price sort)
- Site-wide product search with deep-link highlighting
- Shopping cart and checkout flow
- User registration, login, and profile
- Admin panel for product and user management
- Multi-image products with carousel modal
- Mobile-first responsive layout

## Tech stack

- HTML, CSS, Vanilla JavaScript
- Client-side demo data (`localStorage` / `sessionStorage`)
- No build step

## Run locally

Open `index.html` in a browser, or serve the project root:

```bash
python -m http.server 8000
```
